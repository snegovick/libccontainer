Libccontainer

Various data structures implemented in plain C

View project onGitHub

Description

Why do these esoteric and hardly-ever-used languages like C++ or Java or Python have cool libraries with lots of readily available implementations of data structures like Boost or STL or whatever and such a major language like C has just few such libs ? This projects aim is to change this situation. I hope you will find something useful here, and feel free to contribute!

Examples

Examples can be found at this page: http://uprojects.org/?page_id=17

Setup

cd libccontainer
bash ./bootstrap
./configure --prefix=/usr # if you dont like autotools default (/usr/local)
make; make install

Whats inside

Well tested:

  • Linked list implementation
  • Hash table implementation

Not so well tested:

  • Dynamic storage (malloc/free inside statically allocated buffer)
  • Circular buffer

Cross-compiling

Cross-compiling was tested on Ubuntu 14.04

cd libccontainer
bash ./bootstrap
./configure --build=arm-none-eabi --disable-tests # disable tests for cross-compiling
make
